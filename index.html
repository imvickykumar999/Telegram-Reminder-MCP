Engine Capabilities

Supercharge your AI client with a robust, persistent schedule coordinator.

Voice & Text Input

Speak or type naturally. Send voice notes or conversational text like "remind me in ten minutes to do yoga". The bot transcribes voice and normalizes spoken numbers automatically.

Full CRUD Operations

Complete lifecycle control. Query pending alarms, update schedules, modify task notes, or cancel timers entirely through chat commands.

Reliable Cron Engine

Driven by a bulletproof background scheduler. Retains pending alerts on restarts, prevents duplicate fires, and guarantees dispatch.

Instant Notifications

Receive reminder pings instantly. The bot uses real-time Webhook updates to notify your Telegram account the exact second a scheduled alert is due.

Quick Setup Roadmap

Get your MCP Bot initialized and connected in four simple stages.

1

Install Server

Run standard terminal installation scripts to bootstrap the MCP engine onto your local environment.

2

Get Token

Message `@BotFather` on Telegram, initialize your customized bot name, and obtain the API Token key.

3

Add Settings

Define environment keys, and link the configuration JSON parameters in Cursor/Claude settings.

4

Ask Client

Converse with your AI model, prompt alerts naturally, and let the system handle dispatch notifications.
